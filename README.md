# 파랑새 뮤직 스튜디오

## 🎵 프로젝트 소개

**파랑새 뮤직 스튜디오**는 AI 기술과 창의성이 만드는 음악 공유 플랫폼입니다. 7080세대의 감성을 담아 발라드, 트로트, 국악트로트 등 다양한 장르의 음악을 제작하고 공유합니다.

### 주요 특징

- 🎨 **화려한 디자인**: 7080 감성의 따뜻한 색상과 현대적 UI
- 🎵 **음악 갤러리**: 카테고리별 음악 전시 및 필터링
- ▶️ **음악 플레이어**: 모달 플레이어로 음악 감상
- ❤️ **즐겨찾기**: 좋아하는 음악 저장 기능
- 📝 **블로그**: 음악 제작 과정 및 영감 공유
- 📧 **연락처**: 협업 및 라이센싱 문의
- 📱 **반응형 디자인**: 모든 기기에 최적화

## 🚀 시작하기

### 1. 로컬 서버 실행

```bash
# Python 3 사용
python3 -m http.server 8000

# 또는 Node.js 사용
npx http-server
```

### 2. 브라우저에서 접속

```
http://localhost:8000
```

## 🎯 주요 기능

### 음악 갤러리
- 6개의 음악 카드 표시
- 카테고리별 필터링 (전체, 발라드, 트로트, 국악트로트, 7080음악)
- 호버 시 재생 버튼 표시

### 음악 플레이어
- 모달 형태의 플레이어
- 재생/일시정지 컨트롤
- 이전/다음 곡 네비게이션
- 진행 바 표시
- 다운로드 및 공유 기능

### 즐겨찾기
- LocalStorage를 사용한 로컬 저장
- 새로고침 후에도 유지

### 블로그
- 3개의 블로그 글 예시
- 음악 제작 과정 공유

### 연락처
- 협업 및 라이센싱 문의 폼
- 소셜 미디어 링크
- 이메일 연락처

## 🔧 커스터마이징

### 음악 데이터 추가

`script.js`의 `musicData` 배열에 새로운 음악 추가:

```javascript
const musicData = [
    { id: 1, title: '곡 제목', artist: '파랑새', genre: '장르', category: 'ballad' },
    // 더 추가...
];
```

### 색상 변경

`styles.css`의 CSS 변수 수정:

```css
:root {
    --primary-color: #1e3a8a;
    --secondary-color: #d4a574;
    --accent-color: #fbbf24;
    /* ... */
}
```

### 콘텐츠 수정

`index.html`에서 직접 수정:
- 소개 텍스트
- 블로그 글
- 연락처 정보
- 이메일 주소
